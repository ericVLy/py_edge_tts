# 文本转音频工具 (Edge-TTS GUI)

这是一个基于 Python 构建的轻量级图形界面（GUI）文本转音频（TTS）工具。它集成了微软开源的 `edge-tts` 语音合成引擎，支持数百种多国语言的高质量语音转写，并专为长文本、网页文章朗读及日常音频制作进行了优化。

## 🌟 功能特性

* **网络语音同步获取**：实时从云端动态加载所有支持的微软 Edge 语音类型（包含多国语言及多种性别音色）。
* **智能语音检索**：提供语音关键字实时过滤功能（支持不区分大小写的搜索，如输入 `zh-cn` 即可快速筛选所有中文语音）。
* **多格式文件加载**：
* 支持直接加载标准 `.txt` 文本文件。
* 支持加载 `.html` / `.htm` 网页文件，并能**智能剥离**其中的 JavaScript、CSS 及网页标签，自动提取干净的连贯纯文本。


* **动态加载反馈**：在音频转写与网络传输期间，执行按钮旁会自动展现动态进度条，避免界面出现假死或未响应现象。
* **历史设置记忆**：自动将用户最后一次选择的语音类型和检索关键字保存至本地 `tts_config.json` 配置文件中，下次启动无缝恢复。
* **全平台支持**：支持在 Windows、Linux 和 macOS 系统上运行或编译。

---

## 🛠️ 环境准备与安装

使用本工具前，请确保你的系统中已安装 **Python 3.7+**。

### 1. 克隆或下载本项目

将代码保存到本地文件夹（例如 `tts_tool.py`）。

### 2. 安装依赖库

在终端（或命令提示符）中运行以下命令安装必要的第三方依赖：

```bash
pip install edge-tts beautifulsoup4

```

---

## 🚀 运行与使用

在项目目录下执行以下命令即可启动图形化界面：

```bash
python tts_tool.py

```

### 使用步骤：

1. **搜索与选择语音**：在“搜索语音”框中输入关键字（如 `xiaoxiao` 或 `en-us`），然后在“选择”下拉菜单中决定目标音色。
2. **导入文本**：直接在中央文本框中打字，或者点击 **加载文件 (TXT/HTML)** 导入本地的文本文档或网页。
3. **导出音频**：点击 **转换为 MP3 并保存** 按钮，选择本地保存路径。等待按钮旁的进度条滚动完毕，即可获得高质量的 MP3 格式音频。

---

## 📦 独立可执行文件 (EXE) 打包

如果你需要无需安装 Python 环境即可独立运行的 `.exe` 文件，可以直接从`Release`下载最新版本，也可以使用 `PyInstaller`手动构建。

### 1. 安装 PyInstaller

```bash
pip install pyinstaller

```

### 2. 打包命令

针对本工具的特性（含有 Tkinter 窗口和 BeautifulSoup4 隐藏导入），推荐使用以下命令进行打包：

```bash
pyinstaller -F -w --hidden-import=bs4.builder._htmlparser tts_tool.py

```

*提示：编译完成后，独立的 `.exe` 文件将生成在 `dist/` 目录下。*
